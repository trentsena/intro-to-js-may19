We can also reference HTMLElements(objects) in the dom.

querySelector

we can use querySelector if we want to reference 1 html element object. Our result will be a single html element object.

var myElement = document.querySelector('h1');

let myElement = document.querySelector('#title');

const myElement = document.querySelector('.items');

querySelectorAll

we can use querySelectorAll if we want to reference more than 1 html element object. Our results would be a list of elements

var myElements = document.querySelectorAll('h2');

let myElements = document.querySelectorAll('[group=cars]');

const myElements = document.querySelectorAll('.items');

getElementById

we can use getElementById if we want to reference 1 html element object. Our result will be a single html element object.

var myElement = document.getElementById('title');

getElementsByTagName

we can use getElementsByTagName if we want to reference more than 1 html element object. Our results would be a list of elements

var myElements = document.getElementsByTagName('li');

getElementsByClassName

we can use getElementsByClassName if we want to reference more than 1 html element object. Our results would be a list of elements

var myElements = document.getElementsByClassName('items');
