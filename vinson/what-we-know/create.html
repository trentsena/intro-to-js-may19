There are lots of ways to create things in javascript. Lets take a look.

variables

to create a variable we use a keyword( var, let, const ).

var myNumber = 9;

let myNumber = 9;

const myNumber = 9;

functions

to create a function we use the keyword function and give it a name or without a name it is an anonymous function. 

function someFunction( parameter1, parameter2 ) {
    const myValue = parameter1; // argument is what we call the value of a parameter.
    const myValue = arguments[1]; // argument[1] will be the value of parameter2.
}

Or you can use an anonymous arrow function. 

( parameter1, parameter2 ) => {
    const myValue = parameter1; // argument is what we call the value of a parameter.
    const myValue = arguments[1]; // argument[1] will be the value of parameter2.
}

HTML Elements

to create an HTML Element we have to use the document object method createElement. 

document.createElement("h1");

Creating an HTMLElement does not place it in the web page. we need to appendChild or prepend. 

document.body.appendChild(myVariableHoldingAnH1);

We can also add them inside an element by using strings. 

document.body.querySelector('main').innerHTML = "<div>  <ul>  <li>  </li> </ul> </div>"
